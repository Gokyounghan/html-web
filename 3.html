JavaScript

JavaScript (JS)는 가벼운 인터프리터형 또는 JIT-컴파일형 프로그래밍 언어로, first-class functions를 지원합니다. 주로 웹 페이지를 위한 스크립팅 언어로 알려졌지만, Node.js, Apache CouchDB, Adobe Acrobat처럼 많은 비 브라우저 환경에서도 사용됩니다. JavaScript는 프로토타입 기반의 다중 패러다임 스크립팅 언어로서, 역동적이고, 객체지향형, 명령형 및 선언형(가령 함수형 프로그래밍) 스타일을 지원합니다. 자세한 내용은 JavaScript에 대하여를 참고하세요.
